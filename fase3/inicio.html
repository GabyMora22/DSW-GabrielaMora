Proposito

El proposito de esta actividad es apropiar las tematicas presentadas en la fase 3, donde se realiza la construcción del sitio web y se emplean técnicas de codificación HTML y se aplican estilos CSS, con el objetivo de que esta pagina web sea facil de navegar y amigable para el usuario.

Introduccion

La presente pagina web tiene varias opciones en la opcion temas puede ver los contenidos que se investigaron. La opcion Tema 1 contiene conceptos basicos de Flex Box y en la opcion Tema 2 contiene conceptos basicos de CSS Grid.
En la opcion Recursos Digitales puede ver contenido multimedial relacionados con las tematicas seleccionadas. La opcion Recurso 1 contiene un video donde se realiza un pequeño ejemplo de Flex Box y en la opcion Recurso 2 contiene un video donde se realiza un pequeño ejemplo de CSS Grid.
En cuanto a las opciones de actividad se completará en la próxima fase y por ultimo tenemos la opcion Autor donde se da una pequeña informacion sobre el creador de este sitio web.
